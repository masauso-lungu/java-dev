# java-dev

Repo for Java projects — crash course practice, school exercises, and DSA.

## 📁 Structure

| Folder | Purpose | Status |
|--------|---------|--------|
| [`CrashCourse/`](CrashCourse/) | Java fundamentals crash course | 🟢 Active |
| [`MUM/`](MUM/) | School exercises & assignments | 🟢 Active |
| [`DSA/`](DSA/) | Data Structures & Algorithms | 🔜 Planned |

## Getting Started

```bash
# Compile & run any Java file
javac <Folder>/<File>.java
java -cp <Folder> <ClassName>

# Example
javac CrashCourse/01-hello-world/HelloWorld.java
java -cp CrashCourse/01-hello-world HelloWorld
```

## Requirements

- **Java JDK 17+** (or whichever version you have installed)
- Any IDE (IntelliJ IDEA, VS Code with Java extensions, etc.)
