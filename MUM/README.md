# MUM — School Exercises

Java exercises and assignments for school coursework.

## Structure

Organize exercises by course or module:

```
MUM/
├── README.md
├── CourseModule1/
│   ├── Exercise1.java
│   └── Exercise2.java
├── CourseModule2/
│   └── ...
└── ...
```

## How to Run

```bash
# Compile a single file
javac MUM/CourseModule1/Exercise1.java

# Run
java -cp MUM/CourseModule1 Exercise1
```
